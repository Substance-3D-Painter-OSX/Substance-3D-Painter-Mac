## Overview

<a href="#">Substance 3D Painter</a> dominates game and film texturing pipelines worldwide. The combination of artistic intuitiveness and technical power makes Painter essential for 3D artists creating realistic digital content. Major studios standardize on Painter for texture creation across teams and projects.

The <a href="#">project organization</a> manages complex texturing tasks efficiently. Texture sets group related surfaces. Multiple UV tiles (UDIM) support high-resolution assets. Project templates standardize workflows across teams. Organization features scale from single assets to massive productions.

<a href="#">Material library</a> access provides production-ready starting points. Thousands of materials cover common surfaces. Filter and search find appropriate materials quickly. Materials customize extensively after application. The library accelerates texturing while enabling unique results.

The <a href="#">anchor points</a> system creates dependencies between layers. Reference effects from other layers as mask inputs. Build complex material relationships that update dynamically. Anchor points enable sophisticated material networks without procedural complexity.

<a href="#">Projection painting</a> transfers references onto models. Project photo references directly onto 3D surfaces. Clone stamp tools work in 3D space. Projection capabilities bridge reference gathering and texture creation.

The <a href="#">symmetry tools</a> accelerate character and prop texturing. Mirror painting across symmetry planes. Radial symmetry handles mechanical components. Symmetry features maintain consistency while reducing repetitive work.

<a href="#">Baking</a> generates maps from high-poly sculpts. Normal maps capture sculpt detail for low-poly models. Ambient occlusion, curvature, and position maps drive generators. Integrated baking simplifies the high-to-low workflow.

The <a href="#">scripting API</a> enables automation and customization. Python scripts automate repetitive tasks. Custom tools extend Painter functionality. Pipeline integration connects Painter to studio infrastructure.

<a href="#">Plugin ecosystem</a> expands capabilities through community contributions. Specialty brushes address specific needs. Export plugins target particular engines. The plugin architecture enables continuous capability growth.
